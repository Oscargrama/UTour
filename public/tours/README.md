# Estructura recomendada para contenido visual por tour

Cada tour debe tener su propia carpeta en `public/tours/<tour-id>/`.

## Archivos esperados
- `hero.jpg` (imagen principal del tour)
- `gallery-1.jpg`
- `gallery-2.jpg`
- Opcional: `gallery-3.jpg`, `gallery-4.jpg`, etc.

## Convención sugerida
- Formato preferido: `.webp` (también funciona `.jpg`)
- Tamaño recomendado hero: 1600px de ancho
- Tamaño recomendado galería: 1200px de ancho
- Peso objetivo por imagen: 150KB a 400KB

## Ejemplo
- `public/tours/guatape-private/hero.jpg`
- `public/tours/guatape-private/gallery-1.jpg`
- `public/tours/guatape-private/gallery-2.jpg`
